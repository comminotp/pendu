But du projet : réaliser le jeu du pendu en PHP
- L'ordinateur choisit un mot au hasard (enfin pas encore...)
- L'utilisateur dispose comme indice initial le nombre de lettre composant le mot secret
- L'utilisateur propose une lettre à la fois
- si elle est correcte, elle complète l'indice
- si elle est fausse, l'image du pendu se construit.
